Factory Inventory Management

Data Flow

User filters

FilterBar updates refs in the useFilters composable (warehouse, category, status, month).

View triggers fetch

View calls api.getX(filters). Axios serializes filters into query-string params.

FastAPI handles

Endpoint applies apply_filters + filter_by_month over in-memory lists.

Pydantic validates

Response is shaped by typed models before returning JSON to the client.

Vue renders

Raw data lands in ref(), derived metrics flow through computed(), template re-renders.

Key Patterns

Filter System

  • Four cross-cutting filters: warehouse, category, status, month
  • Quarter shortcuts (Q1-2025…Q4-2025) expand to months server-side
  • Inventory has no time dimension — month filter is ignored there
  • Shared via the useFilters composable

Reactivity Model

  • Raw API data sits in ref() arrays
  • Derived metrics live in computed() (cached)
  • Views own their loading + error state
  • v-for keys use real IDs (sku, month) — never indexes
